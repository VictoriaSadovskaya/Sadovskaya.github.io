   Course description:  This is a course in abstract linear algebra. It emphasizes theory and proofs
   rather than computations. Topics include vector spaces, linear maps, eigenvalues and eigenvectors,
   inner-product spaces, and operators on vector spaces.
   Prerequisite:  Math 311W.

   Text:  Linear Algebra Done Right by Sheldon Axler, 3rd edition.
